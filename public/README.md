# Bookpango - Global Book Review Blog

A fully-featured book review blog built with Firebase, HTML, CSS, and JavaScript.

## Features
- User authentication (Email/Password, Google)
- Admin panel for content management
- Hero slider with admin control
- Book reviews with affiliate links
- Search functionality
- SEO optimized with sitemap and robots.txt
- Responsive design

## Setup
1. Clone the repository
2. Install Firebase CLI: `npm install -g firebase-tools`
3. Login to Firebase: `firebase login`
4. Initialize project: `firebase init`
5. Deploy: `firebase deploy`

## Project Structure
- `public/` - Static files and HTML pages
- `css/` - Stylesheets
- `js/` - JavaScript modules
- `assets/` - Images and fonts

## Firebase Configuration
Update the Firebase config in `js/firebase.js` with your project details.
